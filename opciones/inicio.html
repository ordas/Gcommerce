Gcommerce


La aplicación Gcommerce permitirá llevar a cabo las labores habituales de logística en un comercio cualquiera.

Gcommerce se desarrolla bajo el framework Phonegap, basado en: html5, css y javascript.
Este framework permite que el desarrollo de una app para dispositivo móvil se simplifique, utilizando practicamente las mismas herramientas que en una web clásica. Además, es capaz de generar ejecutables compatibles con cualquiera de las plataformas móviles habituales: Android, IOS, Blackberry, Windows Phone, etc...



Gcommerce utilizará una BBDD MySQL, situada en el cloud de google (https://cloud.google.com/) y por tanto accesible a través de IP desde cualquier lugar, como origen de información principal.

Al utilizar una BBDD externa, la aplicación inicialmente necesitará una conexión a internet para poder comprobar usuario y contraseña. Después podría funcionar sin ella, salvo cuando hubiese que enviar o recibir información de central.

Gcommerce también contendrá en el dispositivo móvil una instancia SQLite para almacenar la información temporalmente hasta su traslado a Central
