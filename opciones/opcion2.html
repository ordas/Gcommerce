Devolución


Opción para hacer una devolución de producto en el comercio.

Contendrá cajas de texto y etiquetas para introducir el producto (opción a lectura con CB) y botón final para finalizar la devolución, y ejecutar el alta del producto nuevamente.


El alta del producto se produce de forma local (dispositivo móvil) hasta que el usuario decida ejecutar la transmisión de datos a la central.
